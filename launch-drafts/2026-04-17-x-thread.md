**Tweet 2/9 — The real distinction**

RAG retrieves paragraphs at query time.

A wiki compiles knowledge into entity pages, concept pages, and cross-references you can actually read.

WikiMem writes the pages once. Every LLM query after that reads structured markdown instead of searching chunks.

(258 chars)

**Tweet 3/9 — Ingest anything**

13+ formats supported: PDF, audio (Whisper), video (ffmpeg), images (Vision), docs, spreadsheets, URLs, Slack exports.

Drop a file. Pipeline detects type, runs the right processor, writes source + entity + concept pages, commits to git.

Same interface for all of them.

**Tweet 5/9 — The self-improving part**

The Observer runs nightly. Scores every page on coverage, consistency, cross-linking, freshness, organization.

Then fixes the worst ones. Capped at ~11 LLM calls and $2 per run so it never runs away.

Knowledge bases decay. This one maintains itself.
